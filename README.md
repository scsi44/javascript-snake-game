# JavaScript Snake Game Workshop

## Vorbereitungen
10min

### Minimal
Alle Dateien von hier herunterladen
Editieren mit Hilfe von "Editor" (oder einem anderen Texteditor)

### Optimal
* Konto auf github erstellen (mit E-Mail Adresse der Schule)
* [git](https://github.com/apps/desktop) installieren
* [vscode](https://code.visualstudio.com/) installieren
* Repo in eigenes github Konto klonen
* Auf Disk Klonen
* Repo öffnen in VisualStudio Code

## Spielbrett erstellen
10min

* Code für "building the board" auskommentieren
* Anhand Code erklären, wie das Spielbrett erstellt wird

## Lass die Schlange erscheinen
15min

* Kommentiere folgenden Code-Teil aus: "snake"
* Erkläre den Code und wie die Schlange erscheint

## Lass das Futter scheinen
10min

* Kommentiere folgenden Code-Teil aus: "food is built"
* Erkläre den Code und wie das Futter erscheint

## Lass die Schlange sich bewegen
15min

* Kommentiere folgenden Code-Teil aus: "the snake to move"
* Erkläre den Code und wie sich die Schlange bewegt

## Lass die Schlange essen
15min

* Kommentieren folgenden Code-Teil aus "for the snake to eat"
* Erkläre den Code und was passiert, wenn die Schlange isst

## Lass die Schlange die Wände berühren
15min

* Kommentieren folgenden Code-Teil aus "snake hit the wall"
* Erkläre den Code und was passiert, wenn die Schlange eine Wand berührt

## Lass die Schlange sich selber berühren
10min

* Kommentiere folgenden Code-Tail aus "the snake hit itself"
* Erkläre, wie der Code funktioniert und was er bewirkt

## Mach, dass sich die Schlange automatisch vorwärts bewegt
10min

* Kommentieren folgenden Code-Tail aus: "automatically move"
* Erkäre, wie sich die Schlange automatisch vorwärts bewegt

## Kompiliere das Spiel und mach es auf dem Internet zugänglich
10min

* github -> settings -> Pages
* deploy from a branch
* deploy from master 

## Weitere Aufgaben
30min

* Mache das Futter grün
* Mach die Schlange grösser
* Erstelle Buttons, um die Richtung der Schlange zu ändern
* Erkläre, was in der Funktion `gameStart.disabled` passiert
* Zeichne das Spielbrett mit Hilfslinien (Reihen und Spalten)
* Steigere die Geschwindigkeit, nachdem die Schlange etwas gefressen hat
* Probiere agentic ai oder eine andere AI aus, um das Spiel zu verändern
